📞 TROUBLESHOOTING QUICK REFERENCE:

Issue                          Solution
─────────────────────────────────────────────────────────────────────────────
Can't connect to backend       Check http://localhost:3001/api/health
API key not working            Verify at https://huggingface.co/settings/tokens
Emulator can't find backend    Use http://10.0.2.2:3001 (already configured)
Port 3001 in use               Change PORT in .env
Node not found                 Install Node.js 16+
Build fails                    Run ./gradlew clean build
Database locked                Restart backend, delete data/app.db
Timeout on first request       Hugging Face loading model (normal, 30-60s)
Out of memory                  Increase Java heap size
Video generation slow          Normal - takes 2-3 minutes

Full troubleshooting: backend/INTEGRATION_GUIDE.md → Troubleshooting
