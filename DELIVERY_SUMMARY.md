## 🚀 How to Get Started

### Step 1: Prerequisites (5 minutes)
```bash
✅ Node.js 16+ installed
✅ pnpm or npm installed
✅ Android Studio installed
✅ Hugging Face API key obtained from:
   https://huggingface.co/settings/tokens
```

### Step 2: Backend Setup (3 minutes)
```bash
cd ai/backend
pnpm install
cp .env.example .env
# Edit .env and add your API key
pnpm run dev
```

Expected output: Server running on port 3001 ✅

### Step 3: Android Setup (2 minutes)
```bash
1. Open ai/ folder in Android Studio
2. Build > Make Project
3. Run > Run 'app'
4. Select emulator
5. App launches ✅
```

### Step 4: Test Features (5 minutes)
- Open "Text Generation" → Enter prompt → See result
- Open "Image Generation" → Enter prompt → See image
- Test remaining 4 features
- All working! ✅

**Total time to working app: ~15 minutes**
