QUICK START
================================================================================

Backend:
  1. cd ai/backend
  2. pnpm install
  3. Add Hugging Face API key to .env
  4. pnpm run dev

Android:
  1. Open ai/ in Android Studio
  2. Update ApiConfig.java with backend URL
  3. Build → Make Project
  4. Run → Run 'app'
  5. See beautiful onboarding flow!
